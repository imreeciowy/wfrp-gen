# howto run
- All you need is generator.py and Being.py in the same folder
- run generator.py x
- where x is:
- d for Dwarf
- e for Elf
- h for Halfling
- u or none for Human
WARNING: Works OK under 3.4.4
I found that 3.4.3 and 3.5.1 have some stability issues.


# to close before end
- ctrl+c kills
